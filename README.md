 WriteMate — Typing & Writing Toolkit

A browser-based toolkit for typing practice and writing. Built as a personal project using vanilla HTML, CSS, and JavaScript — no frameworks, no backend, no sign-up required.

🔗 **Live demo:** [writemates.netlify.app](https://writemates.netlify.app/)

Features

- **Typing Speed Test** — 15, 30, or 60-second tests with real-time WPM, accuracy, and character count
- **Keyboard Practice** — Drills for home row, top row, numbers, and common letter pairs, with an interactive on-screen keyboard
- **Writing Tools** — Word counter, case converter, text transformer, and lorem ipsum generator
- **Essay Templates** — Six ready-made structures including argumentative essay, lab report, and cover letter
- **Built-in Stats** — Words, sentences, paragraphs, and reading-time estimates
- **Works offline** — Everything runs in the browser, no server required

Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Deployed on Netlify

 Running Locally

Clone the repository and open `index.html` in any modern web browser:

```bash
git clone https://github.com/ProfSachin/writemate.git
cd writemate
open index.html
```

No build step or dependencies required.
